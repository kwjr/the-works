Migrating from separate installs?

One-time setup
Each tool stores data in your browser's localStorage, tied to its origin URL. Because this is a new URL, your tools will start fresh — but your old data is safe at the previous addresses. Follow the steps below for each tool to export and re-import your data. You only need to do this once.

Clockwork

  1. Visit your old Clockwork URL
  2. Open Settings → scroll to Data Management
  3. Click Download Backup — saves a .json file
  4. Open Clockwork here and go to Settings → Data Management
  5. Click Restore from Backup and select the file

Busywork

  1. Visit your old Busywork URL
  2. Open Settings (gear icon) → Export Data
  3. Save the .json backup file
  4. Open Busywork here and go to Settings → Import Data
  5. Select the file — all tasks and projects will be restored

Framework

  1. Visit your old Framework URL
  2. Click the ⬇ Download Backup button in the toolbar
  3. Save the .json backup file
  4. Open Framework here and click ⬆ Restore Backup
  5. Select the file — all projects and artifact states will be restored

Groundwork

  1. Visit your old Groundwork URL
  2. Open Settings → click ↓ Save backup
  3. Save the .json backup file
  4. Open Groundwork here and click ↑ Choose file…
  5. Select the file — charter, meetings, actions, and decisions are restored
